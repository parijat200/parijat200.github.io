Sales Analysis

Tools:

Tools and Libraries Used:
  • Python: for data manipulation and analysis.
  • Pandas: for data loading, cleaning, and transformation.
  • Matplotlib and Seaborn: for data visualization.
  • Pyarrow: for Feather file format compatibility.

Objective: The objective of this analysis is to gain insights into the sales data of an electronics store, identify patterns and trends, and provide actionable insights for business decision-making. This includes identifying the best month for sales, the city with the maximum orders, the most sold product, and understanding product bundling.

About The Data: The dataset comprises YouTube comments and videos data, including details like comment text, likes, dislikes, views, category IDs, and more. The data is stored in CSV and JSON formats, with additional data files to be combined for comprehensive analysis.

Python YouTube Analysis

Tools:

Tools and Libraries Used:
  • Python: for data manipulation and analysis.
  • Pandas: for data loading, cleaning, and transformation.
  • Matplotlib and Seaborn: for data visualization.
  • TextBlob: for sentiment analysis.
  • WordCloud: for generating word clouds.
  • Emoji: for emoji extraction and analysis.
  • SQLAlchemy: for database operations.
  • Plotly: for interactive visualizations.

Objective: The objective of this analysis is to explore and analyze YouTube comments and video data to extract meaningful insights. This includes performing sentiment analysis, identifying patterns in positive and negative comments, analyzing emoji usage, and understanding trends in video engagement and popularity.

About the Data: The dataset comprises YouTube comments and video data, including details like comment text, likes, dislikes, views, category IDs, and more. The data is stored in CSV and JSON formats, with additional data files to be combined for comprehensive analysis.
